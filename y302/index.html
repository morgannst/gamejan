กำหนด Google Apps Script Web App URL เพื่อส่งคะแนนบันทึกเข้าไปยัง Google Sheet ไฟล์ ID: 1GrvgmkCiHZgxM9J0K4LFk1HMu7K1_nLBc24YNLSQ6sM (แผ่นงาน: ScoreData)

หลังจากคัดลอก Web App URL จากขั้นตอนการ Deploy ของ Google Sheets แล้ว ให้นำมาวางที่นี่และกดบันทึก

วิธีการติดตั้ง Apps Script ใน Google Sheet

  1. เปิด Google Sheet ไฟล์ของคุณครู
  2. เข้าที่เมนู ส่วนขยาย (Extensions) > Apps Script
  3. คัดลอกไฟล์โค้ด Apps Script ไปวางทับหน้าว่าง แล้วกด บันทึก
  4. กดปุ่ม การทำให้ใช้งานได้ (Deploy) > การทำให้ใช้งานได้ใหม่ (New deployment) เลือกชนิดเป็น เว็บแอป (Web app)
  5. ตั้งค่าสิทธิ์ให้ "ผู้มีสิทธิ์เข้าถึง (Who has access)" เป็น "ทุกคน (Anyone)" จากนั้นนำ URL ที่ได้มาป้อนที่นี่
